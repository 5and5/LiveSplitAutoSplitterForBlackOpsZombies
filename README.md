## using
in livesplit:<br>
Open Splits > From File > bo1 blank splits.lss<br>
Open Layout > From File > zombies_layout.lsl<br>
Edit Layout > double click Scriptable Auto Splitter > Set Script path to path of autosplit_bo1zombies.asl<br>
Ok all<br>
Save the Layout<br>
